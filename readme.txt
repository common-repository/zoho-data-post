 It's possible to post the data from WP to Zoho CRM using the Zoho CRM's API.  It can choose which is the leads tab or the contacts tab in Zoho CRM.   

== Description ==
1. You can login Zoho CRM.
2. and then get the Zoho API Key. 
3. login Word Press and check "Settings" => "Zoho CRM Data Post"
4. paste 1) API Key in the form of Zoho CRM API Key. 
5. enter in the last name and first name in the field and the click the Post. 
6. The date post to Zoho CRM (you can choose Leads or Contacts tab)

== Installation ==
1. Upload the "zoho-data-post" to the "/wp-content/plugins/\" directory.
2. Activate the plugin through the \"Zoho CRM Data Post\" Setting menu in WordPress.
